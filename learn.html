Form 1 Algebra Notes

1. Algebraic Expressions

Algebraic expressions consist of numbers, variables, and arithmetic operations. Examples:

  • 3x + 5
  • 4a - 7
  • 2m² + 3m - 5

Simplifying Algebraic Expressions: Combine like terms by adding or subtracting their coefficients. Example: 3x + 5x = 8x.

2. Algebraic Operations

Addition and Subtraction: Combine like terms by adding or subtracting coefficients. Example: 7x - 3x = 4x.

Multiplication: Multiply coefficients and variables separately. Example: 3x × 2y = 6xy.

Division: Divide coefficients and cancel common variables. Example: 8x² / 4x = 2x.

3. Linear Equations

A linear equation is an equation of the form ax + b = c. To solve it, isolate the variable by performing inverse operations. Example: 3x + 5 = 20 → x = 5.

4. Substitution

Substitution involves replacing variables with their given values to find the result. Example: Given x = 4, find the value of 3x + 7 → 19.

5. Factoring Algebraic Expressions

Factoring means expressing an algebraic expression as a product of factors. Example: x² + 5x = x(x + 5).

6. Expanding Brackets

Expanding involves multiplying each term inside the bracket by the factor outside. Example: 2(x + 3) = 2x + 6.

7. Problem Solving with Algebra

Algebra helps in solving real-world problems by forming equations. Example: The cost of 3 kg of apples at (5x + 3) per kg is 3(5x + 3) = 15x + 9.
