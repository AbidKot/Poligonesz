Form 1 Algebra Notes

1. Algebraic Expressions: Mathematical phrases that include numbers, variables, and operations. Examples: 3x + 5, 4a - 7.

2. Simplifying Algebraic Expressions: Combine like terms by adding or subtracting coefficients. Example: 3x + 5x = 8x (Combine 3 and 5).

3. Linear Equations: Solve equations like ax + b = c by isolating the variable. Example: 3x + 5 = 20 â†’ x = 5.

4. Factoring: Express as a product of factors by bringing out the common term. Example: x^2 + 5x = x(x + 5).

5. Expanding Brackets: Multiply each term inside the bracket by the factor outside. Example: 2(x + 3) = 2x + 6.

6. Substitution: Replace variables with values to calculate the result. Example: If x = 4, then 3x + 7 = 19.

7. Problem Solving: Use algebra to model and solve real-world problems.
